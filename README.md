# Pitagorax.com
Web de la empresa

# Instalación
1. Descargar o clonar el repositorio.
2. Ejecutar "npm install" en el directorio del proyecto.
3. Ejecutar "ng serve --open".
4. Se abrirá una ventana del navegador con la web.
